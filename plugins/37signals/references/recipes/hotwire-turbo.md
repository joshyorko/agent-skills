# Hotwire Turbo

Use Turbo Drive, Frames, Streams, and morphing to keep rendering, permissions,
and validation server-owned while delivering useful interaction.

Do: choose the smallest Turbo primitive, test stale and failed submissions, and
keep frame boundaries domain-oriented.

Avoid: replacing simple server HTML with client state by default.
