# Shape Up Pitch

Use this order: problem, appetite, solution sketch, boundaries, rabbit holes,
no-gos, circuit breaker, and recommendation.

Do: keep scope variable and appetite fixed. Make the bet understandable without
a task list.

Avoid: backlog dumps, premature implementation plans, and automatic extensions.
