# Product Copy

Treat words as interface. Labels, button text, validation messages, empty states,
and confirmation copy should remove uncertainty and use domain language.

Do: use plain verbs and nouns from the work, cut generic SaaS abstractions, and
make irreversible actions specific.

Avoid: cleverness, filler, and copy that explains implementation details.
