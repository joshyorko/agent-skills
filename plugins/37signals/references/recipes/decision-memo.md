# Decision Memo

Use this for async decisions. Include context, options, tradeoffs, recommendation,
owner, deadline if any, and what happens next.

Do: separate facts from judgment and state the decision needed in one sentence.

Avoid: consensus theater, vague next steps, and meetings that only restate the
memo.
